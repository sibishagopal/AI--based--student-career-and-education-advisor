# AI-Based Student Career and Education Advisor

## Project Overview
The AI-Based Student Career and Education Advisor is a web application that helps students choose suitable careers and educational paths based on their interests, skills, aptitude, and academic performance. The system provides personalized career recommendations, course suggestions, college recommendations, and learning guidance.

## Features
- Student Registration and Login
- Student Dashboard
- Aptitude Test
- Skills and Interest Assessment
- AI-Based Career Recommendation
- College and Course Recommendation
- Resume Analyzer
- Career Chatbot
- Student Profile Management
- Admin Panel

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP
- Version Control: Git and GitHub

## Installation
1. Install XAMPP.
2. Copy the project folder to `C:\xampp\htdocs\`.
3. Start Apache and MySQL.
4. Create a MySQL database named `career_advisor`.
5. Import the SQL file into the database.
6. Open the project in your browser using:
   `http://localhost/AI-Based-Student-Career-and-Education-Advisor`
